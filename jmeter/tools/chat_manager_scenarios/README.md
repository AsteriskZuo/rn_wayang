# ChatManager Scenario JMX Generator

The plans in `jmeter/data/chat-manager/` are generated by running:

```bash
node jmeter/tools/chat_manager_scenarios/generate.js
```

Run the generator test before committing generated `.jmx` changes:

```bash
node --test jmeter/tools/chat_manager_scenarios/generate.test.js
```

Each scenario lives in `jmeter/tools/chat_manager_scenarios/scenarios/` so the
business sampler chains can be implemented independently. Keep shared JMX,
WebSocket, assertion, and JSR223 helpers in `generate.js`.

Generated plans intentionally use normal ordered Thread Groups. Do not move
init, login, scenario samplers, or logout into setUp or tearDown Thread Groups.
